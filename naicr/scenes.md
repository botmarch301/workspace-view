# 싼 월세집은 나 혼자가 아니었다 — 씬 목록
> **Status**: 3차 개정 | **Created**: 2026-02-22 | **Revised**: 2026-03-29
> **참조**: [concept.md](concept.md) · [scenario.md](scenario.md)

---

## 범례

| 타입 | 설명 |
|------|------|
| **VN** | 비주얼 노벨 연출 (대사, 회상, 선택지) |
| **SIM** | 생활 시뮬레이션 (일상 관리) |
| **ARC** | 아케이드 인터랙션 (QTE, 리듬) |
| **SEX** | 성감/성교 시뮬레이션 |
| **END** | 엔딩 씬 |

> **씬 번호는 정리용 인덱스**. 게임 내 실제 진행은 SIM 루프 중 조건에 따라 이벤트가 트리거되는 구조이며, 번호 순서대로 강제 진행되지 않음.

---

## 막 경계

| 전환 | 트리거 |
|------|--------|
| **1막 → 2막** | VR 구입 (공포 도피 수단 획득) |
| **2막 → 3막** | 벽에 숨은 귀신에게 삽입 → 귀신 도망 (주인공이 귀신을 인지) |
| **3막 → 4막** | 첫 보름달 성교 (성적 관계 성립) |

---

## 1막: 공포 (기)

> 순수 공포. 악귀(검은 그림자)의 정신 공격에 속수무책으로 당하는 단계.

| # | 씬 이름 | 타입 | 설명 |
|---|---------|------|------|
| 1-01 | 이사 & 정착 | VN | 새 집에 도착, 집 상태 확인, 싼 이유에 대한 불안 |
| 1-02 | 첫 출근 | VN | 게임 회사 출근, 동료 소개 |
| 1-03 | 밤의 괴현상 | VN | 소름끼치는 현상 발생 (소리, 그림자, 온도 변화) |
| 1-04 | 가위눌림 | ARC | 수면 중. 귀신 접근 → 좌/우 연타로 몸 풀기. 피해 누적형 |
| 1-05 | 고개 돌리기 | ARC | 일상 중. 소리 → 시선 끌림 → 반대 방향키로 저항. 즉시 판정형 |
| 1-E1 | **심장마비 엔딩** | END | 공포 과다 → 심장마비 사망 |
| 1-E2 | **악귀 회귀 엔딩** | END | 이사 결심 → 집은 다시 악귀의 집으로 |

---

## 2막: 정화의 시작 (승)

> VR로 공포 차단 → 자위 → 정액 노출 → 귀신 복원 → 발각 → 삽입.

| # | 씬 이름 | 타입 | 설명 |
|---|---------|------|------|
| 2-01 | VR 구입 | VN/SIM | 공포 도피 수단. 인터넷 검색(괴담)도 이 즈음에 연출로 발생 |
| 2-02 | 첫 VR 자위 → 정액 노출 | VN | 악귀가 정액에 묻음 → 아랫배(자궁) 복원. 괴현상 줄어듦 |
| 2-03 | VR 자위 일상 (반복) | SIM | 반복 자위. 귀신 신체 점진적 복원 (플레이어만 봄) |
| 2-04 | 손가락 빠는 버릇 → 젖 물리기 | VN | 젖가슴 복원 후, 잠결에 손가락 빠는 주인공에게 젖을 물림 (주인공 모름) |
| 2-05 | 젖 물리기 반복 | VN | 매일 반복. 귀신 표정 미세 변화 — 성적 만족인지 모성애 충족인지 애매 (플레이어만 봄) |
| 2-06 | 만월 — 69자세 | VN/SEX | 평소와 다르게 69자세 → 귀신 성기를 만지고 빨아줌 → 귀신 놀라서 도망 |
| 2-07 | 벽에 숨는 귀신 → 삽입 | VN/SEX | 벽으로 숨지 못한 하체에 삽입 → 귀신 놀라서 도망. **2막 끝** |

---

## 3막: 접근 (전)

> 귀신과의 거리를 좁히는 과정. 스킨십, 제물, 보름달 성교.

| # | 씬 이름 | 타입 | 설명 |
|---|---------|------|------|
| 3-01 | 귀신 도망 후 — 안 나타남 | VN | 부르고 싶지만 반응 없음 |
| 3-02 | 젖 빨기 (자는 척) | ARC | 알면서 빠는 인터랙션. 혀 좌우 조작. 들키지 않는 범위에서 최대 자극 |
| 3-03 | 팜플렛 사건 | VN | 결혼/출산 팜플렛을 귀신이 빤히 쳐다보다 들킴 |
| 3-04 | 불 트라우마 발견 | VN | 우연히 불 → 귀신의 발작적 거부. 이후 라이터를 방어 수단으로 사용 가능 |
| 3-05 | **첫 보름달 — 첫 성교** | SEX | 발정 → 핥고 빨기 → 삽입 → "임신 시켜달라" → 질내사정. **3막 끝** |
| 3-E1 | **복상사 엔딩** | END | 동료 부적 → 귀신 차단 → 보름달 성욕 폭발 → 사망 |
| 3-E2 | **퇴마 (상담) 엔딩** | END | 상담 → 처녀귀신 진단 → 퇴마 부적 → 퇴치 |
| 3-E3 | **퇴마 (퇴마사) 엔딩** | END | 퇴마사 호출 → 귀신 퇴치 |

> **제물 놓기**: 별도 씬이 아닌 SIM 시스템 (인벤토리 → 단상 배치). 귀신이 제물에 주의를 뺏긴 틈에 스킨십 가능.

---

## 4막: 성감 개발과 성불 (결)

> 양기 관리 + 성감 개발 + 귀신의 사연 해명 + 성불.

| # | 씬 이름 | 타입 | 설명 |
|---|---------|------|------|
| 4-01 | 귀신 등장 일상화 | VN | 보름달 이후 밤마다 나타남 (라이터 켰으면 2~3일 후). 성교 or 수면 중 양기 빨림 |
| 4-02 | 성교 이후 — 삽입 시도 | SEX | 귀신이 잠든 주인공에게 삽입 시도 → 실패 → 깨서 다시 성교 |
| 4-03 | 성욕 에스컬레이션 (반복) | VN/SEX | 바지 내림, 공중 삽입 등 점점 과격해지는 귀신 |
| 4-04 | 코피 — 양기 부족 경고 | VN | 회사에서 코피. "이러다 죽겠다" |
| 4-05 | 성감 개발 시작 (VR 야동) | VN/SEX | VR 야동 구매 → 귀신과 동일한 시스템으로 작동 (암묵적 튜토리얼) |
| 4-06 | **성감 개발 (반복)** | SEX | VR에서 익힌 것을 귀신에게 적용. 앵커 시스템 인터랙션 |
| 4-07 | 귀신의 호의 시작 | VN | 청소해줌, 위험에서 보호, 가슴 물려줌 |
| 4-08 | 출장 | VN | 회사 출장 발생 |
| 4-09 | **노녀 구출** | VN | 출장 귀환길에 사고 현장에서 투신하려는 노녀를 말림 |
| 4-10 | **남편 귀신 성불** | VN | 남편 귀신이 따라옴 → 아내를 보고 깨달음 → 성불 (귀신/플레이어만 봄) |
| 4-11 | 아내의 거부감 해소 | VN | 남편 성불을 지켜본 후 주인공에 대한 감정이 열림 |
| 4-12 | **마지막 보름달 — 성교** | SEX | 발정하지 않음. 평상시처럼 성교. 한이 풀렸음을 체감 → 성불 |
| 4-E1 | **트루 엔딩 (성불)** | END | 임신 확인 → 한이 풀리고 성불 |
| 4-E2 | **양기 고갈 엔딩** | END | 성감 개발 부족 or 지박령 성불 미완 → 양기 소진 사망 |

## 필요 배경

| # | 배경 | 사용 빈도 | 비고 |
|---|------|----------|------|
| 1 | **집 — 뷰 A (침대 쪽)** | ★★★★★ | 수면, VR 자위, 성교, 젖 물리기. 밤 씬의 메인 뷰 |
| 2 | **집 — 뷰 B (책상/현관 쪽)** | ★★★☆ | 식사, 제물 단상, 귀신 목격. 일상 씬의 메인 뷰 |
| 3 | **집 — 뷰 C (전체/창문 쪽)** | ★★☆ | 이사, 청소, 귀신 호의. 낮 씬·분위기 전환용 |
| 4 | **회사** | ★★☆ | 출근, 동료 대화, 코피, 출장 |
| 5 | **거리/상점** | ★☆☆ | 출퇴근 이동 |
| 6 | **사고 현장** | ★☆☆ | 노녀 구출, 남편 귀신 성불 (4막 핵심) |
| 7 | **핸드폰 화면** | ★★☆ | UI 오버레이 (쇼핑, 배달, 시간떼우기 + 이벤트 시 검색/상담 연출) |
| 8 | **VR 공간** | ★★☆ | 가상 공간 (VR 자위/야동 내부) |

> **실제 배경 6개** (집 3뷰 + 회사 + 거리 + 사고현장) + UI 1개 + 가상 1개 = **총 8개**
> 회상 배경 삭제 — 귀신의 사연은 성불 시 연출로 처리.
