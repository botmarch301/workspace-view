## 설계 원칙

1. **"보여주되 설명하지 않는다"** — 내레이션보다 시각/청각 연출로 전달
2. **플레이어 시점 vs 관객 시점** — 주인공이 모르는 것을 플레이어가 보는 극적 아이러니 활용
3. **공포는 보이지 않는 것에서 온다** — 직접적인 점프스케어보다 암시와 분위기

## 1. 시점 연출 체계

### 1-1. 이중 시점 구조

이 게임의 핵심 연출은 **주인공이 모르는 것을 플레이어가 목격하는** 구조다.

| 시점 | 설명 | 사용 상황 |
|------|------|----------|
| **주인공 시점 (1st)** | 주인공이 보고 느끼는 것만 표현 | 일상, 공포 인터랙션, 선택지 |
| **관객 시점 (3rd)** | 주인공이 모르는 장면을 플레이어에게 보여줌 | 귀신의 행동, 정화 과정, 밤의 비밀 |

### 1-2. 시점 전환 규칙

| 전환 | 연출 | 예시 |
|------|------|------|
| 주인공 → 관객 | 주인공이 잠들거나 의식을 잃은 후 카메라가 빠지듯 이동 | 수면 후 귀신의 젖 물리기 |
| 관객 → 주인공 | 알람/소리로 주인공이 깨어남 + 카메라가 1인칭으로 복귀 | 아침 기상 |
| 관객 시점 중 주인공 반응 | 주인공의 무의식적 반응(미소, 빨기)만 보여줌 | 젖을 빠는 주인공의 평화로운 표정 |

### 1-3. 막별 시점 비율

| 막 | 주인공 시점 | 관객 시점 | 비고 |
|----|-----------|----------|------|
| 1막 | 90% | 10% | 관객 시점은 젖 물리기 장면에만 사용 |
| 2막 | 60% | 40% | 귀신의 복원 과정을 관객이 목격 |
| 3막 | 70% | 30% | 밤의 비밀 + 보름달 접근 |
| 4막 | 80% | 20% | 대화 가능 후 대부분 공유 시점. 남편 귀신 성불만 관객 시점 |

## 2. 카메라 연출

### 2-1. 기본 카메라 세팅

| 상황 | 카메라 | 효과 |
|------|--------|------|
| **일상 (SIM)** | 고정 와이드샷 | 안정감, 일상의 반복 |
| **대화 (VN)** | 미디엄샷, 캐릭터 중심 | 감정 전달 |
| **공포 (ARC)** | 1인칭 / 더치앵글 | 불안감, 통제력 상실 |
| **성적 장면 (SEX)** | 클로즈업, 슬로우 패닝 | 친밀감, 집중 |
| **회상** | 고정 와이드, 약간 높은 앵글 | 기억의 객관성, 거리감 |

### 2-2. 카메라 언어

| 기법 | 의미 | 사용 예 |
|------|------|--------|
| **더치앵글** (기울어진 앵글) | 불안, 공포, 세계가 기울어진 느낌 | 가위눌림, 귀신 첫 목격 |
| **로우앵글** (아래에서 위로) | 위압감, 공포 대상의 크기 강조 | 악귀가 내려다보는 장면 |
| **오버더숄더** | 관계성, 대화의 친밀도 | 4막 대화 장면 |
| **POV (1인칭)** | 몰입, 체험 | 공포 인터랙션, 성감 시뮬 일부 |
| **풀샷 → 클로즈업 줌** | 감정 고조, 발견의 순간 | 귀신 얼굴이 선명해지는 순간 |
| **고정 카메라 + 피사체 이동** | 긴장감, 무언가 다가오는 느낌 | 어둠 속에서 귀신이 서서히 접근 |

## 3. 공포 연출 문법

### 3-1. 공포 3단계

| 단계 | 기법 | 예시 |
|------|------|------|
| **암시** | 환경의 미세한 변화 (소리, 온도, 물건 위치) | 아침에 물건이 옮겨져 있음, 냉기 |
| **인지** | 존재를 감지하지만 확인할 수 없음 | 인기척, 시야 끝의 그림자, 거울 반사 |
| **직면** | 대상을 직접 목격 | 가위눌림 중 악귀 목격, VR 벗었을 때 발각 |

### 3-2. 공포 타이밍 원칙
- **기대를 만들고 비틀기** — 공포가 올 것 같은 순간에 안 오고, 안심한 순간에 온다
- **정적의 활용** — 공포 직전 모든 소리를 멈춤 (1~2초 완전 무음) → 충격 효과 극대화
- **반복과 변주** — 같은 패턴을 2~3회 반복한 후 4회째에 변화를 줌
  - 예: 밤마다 같은 소리 → 같은 소리 → 같은 소리 → 소리 없이 귀신이 바로 옆에

### 3-3. 시각적 공포 연출

| 기법 | 설명 | 빈도 |
|------|------|------|
| **화면 가장자리 침식** | 검은 그림자가 화면 테두리에서 안쪽으로 침범 | 공포 누적 시 상시 |
| **서브리미널 프레임** | 1~2프레임만 귀신 이미지 삽입 (의식 못 하지만 불안감 유발) | 1막 중반부터 간헐적 |
| **조명 깜빡임** | 방 조명이 불규칙하게 점멸 | 귀신 접근 시 |
| **화면 노이즈** | VHS 노이즈/글리치 효과 | 악귀 상태 직면 시 |
| **숨결 안개** | 화면에 숨결 같은 안개가 서림 | 귀신이 가까이 있을 때 |
| **전자담배 빛의 흔들림** | 유일한 광원인 LED가 떨림 | 공포 인지 단계 |

### 3-4. 불 트라우마 연출 (특수)
- 향초/라이터 → 귀신의 얼굴이 **1프레임 동안 Phase 1(악귀)로 회귀**
- 화면 전체에 **붉은 플래시** + 비명 (날카로운 고주파)
- 이후 귀신 사라짐 + 2~3초간 화면 떨림

## 4. 감정 연출 문법

### 4-1. 귀신의 감정 표현 단계별 제약

| Phase | 표현 수단 | 제약 |
|-------|----------|------|
| Phase 1 (악귀) | 움직임, 분위기만 | 표정 없음 (해골), 음성 없음 |
| Phase 2 (부분 복원) | 몸의 반응, 동작 | 얼굴 흐릿, 단편적 소리만 |
| Phase 3 (완전 복원) | 표정, 음성, 대사 | 제약 없음 |

### 4-2. 주인공의 감정 표현

| 감정 | 시각 연출 | 청각 연출 |
|------|----------|----------|
| 공포 | 화면 떨림, 시야 축소, 땀 이펙트 | 심장 박동, 거친 호흡 |
| 안도 | 밝기 회복, 시야 확대 | 한숨, 배경음 복귀 |
| 흥분 | 화면 색조 따뜻하게, 약간의 블러 | 호흡 빨라짐 |
| 슬픔 | 채도 저하, 느린 카메라 이동 | 피아노/현악 저음 |
| 놀람 | 줌인 + 화이트아웃 (짧게) | 임팩트 효과음 |

## 5. 씬별 연출 플로우

### 5-1. 1막: 공포의 확립

**연출 키워드**: 고립, 불안, 도피

```
[이사] 밝은 낮 → 좁은 방의 현실 → 첫 밤이 되며 급격히 어두워짐
  ↓
[괴현상] 소리만 → 그림자만 → 온도만 (직접 보여주지 않음)
  ↓
[가위눌림] 갑작스러운 1인칭 전환 → 연타 인터랙션
  ↓
[VR 도피] VR 공간은 밝고 안전 → 현실과의 대비 극대화
  ↓
[젖 물리기 시작] 관객 시점 전환 — 잠든 주인공 + 다가오는 악귀
  → 긴장감 속에서 의외로 부드러운 행위 → 공포와 모성의 기묘한 공존
```

### 5-2. 2막: 극적 아이러니

**연출 키워드**: 관음, 긴장, 발각

```
[VR 자위] 주인공 시점: "VR이 실감나네" / 관객 시점: 귀신이 빨고 있음
  → 화면 분할 없이, 씬 전환으로 두 시점을 교차
  ↓
[복원 과정] 관객만 목격 — 악귀 → 여성으로 변하는 과정
  → 느린 디졸브로 외형 변화 강조
  ↓
[발각] VR을 벗는 동작 → 0.5초 정지(프리즈 프레임) → 주인공의 눈 클로즈업
  → 가장 임팩트 있는 순간. 음악 완전 정지 후 1초 뒤 반응
  ↓
[거리 좁히기] 구석의 여자 → 벽에 숨기(코미디 요소) → 접촉 → 사라짐
  → 공포에서 호기심으로 감정 톤 전환
```

### 5-3. 3막: 유혹과 첫 교감

**연출 키워드**: 탐색, 발정, 결합

```
[제물 시스템] SIM 화면에서 아이템 배치 → 시간 경과 → VN으로 귀신 반응
  → 반응 없는 제물: 무변화 / 반응 있는 제물: 카메라가 귀신의 반응을 잡음
  ↓
[보름달 접근] 달 아이콘 변화 + 밤 씬의 색조가 점점 푸르게
  → 귀신의 자제력 약화: 사라지는 속도 느려짐 → 눈 마주침
  ↓
[첫 보름달 — 첫 성교]
  → 창문으로 보름달 빛 유입 (은백색 조명)
  → 귀신이 먼저 다가옴 — 카메라가 천천히 클로즈업
  → "임신 시켜달라" — 이 대사만 선명한 음성으로, 나머지는 무음
  → 해가 뜰 때까지: 시간 경과를 하늘 색 변화로 표현 (푸른 밤 → 붉은 새벽)
```

### 5-4. 4막: 관계와 해소

**연출 키워드**: 유대, 발견, 이별

```
[성감 개발] SEX 모드 — 터치와 반응의 피드백 루프
  → 성공할수록 화면 색조가 따뜻해짐 (냉색 → 난색 전환)
  ↓
[얼굴 선명화] 여러 씬에 걸쳐 점진적으로
  → 처음: 윤곽만 → 중간: 눈/입이 보임 → 최종: 완전한 얼굴
  → 완전히 보이는 순간 — 카메라 슬로우 줌인 + 주인공과 눈 마주침
  ↓
[회상 씬] 세피아 + 물결 디졸브로 진입
  → 과거의 행복한 장면 → 방화 장면 (갑작스러운 붉은 색조 전환)
  → 회상 중 귀신의 현재 표정을 간헐적으로 삽입 (과거와 현재 교차)
  ↓
[남편 귀신 성불] 관객 시점 — 주인공은 모름
  → 사고 현장에서 따라온 남편 귀신이 아내를 봄
  → 두 귀신 사이의 무언의 교감 (대사 없이, 표정과 빛으로만)
  → 남편이 빛 속으로 사라짐 → 아내의 눈물
  ↓
[트루 엔딩 — 성불]
  → 보름달 빛 + 임신 확인 (입덧, 젖)
  → 귀신이 서서히 빛나기 시작
  → 마지막 미소 → 빛 입자로 흩어짐
  → 빈 방에 남은 주인공 — 롱 테이크, 긴 정적
  → 창문으로 아침 햇빛이 들어옴 (게임 전체에서 처음으로 환한 자연광)
```

| 회사 | 형광등 | 5000K | 일상, 평범 |

### 6-2. 감정별 조명 변화

| 감정/상태 | 조명 변화 |
|----------|----------|
| 공포 증가 | 광원 범위 축소, 그림자 깊어짐 |
| 안도/안정 | 광원 범위 확대, 부드러운 확산광 |
| 성적 긴장 | 따뜻한 색조 + 낮은 콘트라스트 |

| 성불 | 점진적 밝아짐 → 화면 전체가 따뜻한 빛으로 채워짐 |

## 7. 사운드 연출 원칙

> 상세 사운드 디자인은 별도 문서로 분리 예정. 여기서는 연출과 결합되는 원칙만 정의.

### 7-1. 레이어 구조

| 레이어 | 설명 | 예시 |
|--------|------|------|
| **환경음** | 항상 깔림, 장소의 실재감 | 냉장고 윙, 시계 째깍, 바람 |
| **BGM** | 감정 톤 설정 | 앰비언트, 피아노, 불협화음 |
| **효과음** | 이벤트에 반응 | 문 삐걱, 발소리, 물건 떨어짐 |
| **음성** | 캐릭터 발화 | 대사, 신음, 비명, 속삭임 |

### 7-2. 핵심 사운드 규칙
- **귀신의 접근 = 환경음 감쇠** — 귀신이 가까울수록 냉장고 소리 등이 줄어듬 (비자연적 정적)
- **정적 → 임팩트** — 공포 직전 모든 레이어 뮤트 → 효과음 하나로 충격
- **전자담배 LED와 연동** — 주인공이 빨 때 짧은 흡입음, 내뱉을 때 긴 배기음 (ASMR)
- **심장 박동** — 공포/흥분 시 diegetic 사운드로 심박 삽입 (BPM 상승 = 긴장도 상승)
- **귀신의 음성 진화**: 
  - Phase 1: 무음 (존재만으로 공포)
  - Phase 2: 비언어 소리 (숨소리, 흐느낌)
  - Phase 3: 속삭임 → 대화 가능
