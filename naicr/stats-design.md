## 설계 원칙

1. **수치 비노출** — 모든 내부 수치는 플레이어에게 직접 표시하지 않음 (소지금 제외)
2. **환경 피드백으로 전달** — 캐릭터 외형/배경 변화/귀신 반응으로 상태를 체감
3. **정규화** — 모든 수치는 0~100 스케일. 계산은 내부에서 소수점 처리
4. **현실성** — 소지금/지출은 현실 시세 기준으로 설정하여 경제적 압박감 구현

## 1. 시간 체계

### 1-1. 하루 구조
| 시간대 | 가능한 행동 |
|--------|-----------|
| **아침** | 세신, 식사, 출근 |
| **낮** | 출근(근무/야근), 핸드폰 |
| **저녁** | 퇴근, 식사, 쇼핑(배달), 청소 |
| **밤** | VR, 핸드폰, 스킨십, 성감 개발, 게임, 음악 감상, 명상 |
| **심야** | 수면 → 가위눌림/젖 물리기/몽정 이벤트 발생 가능 |

- **행동 1회 = 1턴**. 시간대별 2~3턴 소화 가능
- 하루 총 **8~12턴** (행동 선택에 따라 유동)
- 수면은 하루의 마지막. 수면 스킵 불가

### 1-2. 보름달 주기
- **28일 = 1주기** (현실 달 주기와 동일, 별도 설명 불필요)
- 14일째 = 보름달: 귀신 성욕 최대 → 성교 이벤트 트리거
- 4막 마지막 보름달이 성불/양기고갈 분기점

## 3. 귀신 수치

### 3-1. 호감도 (AFFECTION) — 행동 해금 조건

| 항목 | 값 |
|------|-----|
| **범위** | 0~100 |
| **초기값** | 0 (1막), 귀신 복원 진행에 따라 상승 |

| 행동 | 변동 |
|------|------|
| 자연 증가 | +0.5/일 |
| 젖 물리기 | +1~2/회 |
| 제물 배치 (선호) | +3~5 |
| 팜플렛 읽는 중 스킨십 | +3~5 |
| 비성적 스킨십 | +2~3 |
| 성적 스킨십 (비성기) | +1~2 |
| 성교 후 | +3~5 |
| 싫어하는 부위 | -3~-5 |
| 라이터 사용 | -10 |

**해금 조건:**
| 호감도 구간 | 가능한 행동 |
|-----------|-------------|
| 0~29 | 접촉 거부 |
| 30~49 | 비성적 스킨십만 |
| 50~69 | 유방/허벅지 접촉 |
| 70+ | 성기 접촉 |

**특수 규칙:**
- 가슴은 2막 젖물리기 이후 호감도 20 이상에서 거부 없음
- 성기 터치 실패시: 격노가 아닌 "민망해서 도망감", 호감 -1~-2, 당일 스킨십 불가

### 3-2. 만족도 (SATISFACTION) — 한의 해소도

| 항목 | 값 |
|------|-----|
| **범위** | 0~100 |
| **성불 조건** | 85~90 이상 + 남편 성불 완료 + 보름달 |
| **의미** | 의사 임신 진행도 |

**상승 경로:**
| 행동 | 변동 |
|------|------|
| 성적 충족 (절정) | +10~+20 |
| 성적 충족 (절정 없이) | +2 |
| 스킨십 | +1~2 |
| 함께 수면 | +2~3 |
| 제물 | +2~3 |
| 남편 성불 | +20 |
| 젖물리기 | +3~5 |

**하락 조건:**
| 행동 | 변동 |
|------|------|
| 보름달 성교 거부/실패 | -10~-15 |
| 5일 이상 방치 | -1/일 |
| 라이터 사용 | -5 |

**중요 사항:**
- 남편 성불 후에도 귀신 행동은 달라지지 않음
- 귀신이 "왜 원하는지 알게 된 것"일 뿐
- 막연한 성욕으로 나아지는 느낌 → 남편 방문 후 기억 회복 → "아이를 원했던 거구나" 자각

## 6. 귀신 심리 흐름 (시나리오 연동 메모)

### 6-1. 귀신 배경 설정
- 고아 출신, 평생 가족에 대한 갈망을 품고 살았음
- 남편과 결혼했으나 주변에서 인정받지 못하는 상황
- 지원 없이 어렵게 살림을 차린 신혼부부 — 이 집이 첫 보금자리
- 집에 화재 발생 → 여자는 뱃속의 아이와 함께 사망
- 남편은 급히 집으로 오다가 교통사고로 따로 사망 (같은 날, 다른 장소)
- 한의 본질: 한 번도 제대로 가져본 적 없는 "가족"이라는 개념에 대한 갈망
- 젖이 나오는 이유: 모성애의 초자연적 표현 (물리적 이유가 아닌 한에서 비롯)
- 젖 물리기/성교는 기억 없이 본능적으로 하는 것, 왜인지 모르지만 나아지는 느낌

### 6-2. 남편 원혼과의 관계
- 남편(지박령)은 질투심이 있지만, 주인공이 노모를 도운 은인이라는 사실을 알고 있어 악감정 없음
- 주인공은 할머니를 도운 것을 기억도 못 하는 사소한 일로 여김
- 남편 성불은 귀신의 한 가지 미련을 해소하는 역할

### 6-3. 막별 심리 변화
**1막**: 원혼 상태, 기억 없음, 공포와 분노만 존재

**2막**: 복원 과정 (기억 없음, 막연한 거리감), 친밀감 증가, 만월 오랄 시도

**3막**: 친밀감 증가, 만월 성교. 기억 없이 성교하면 막연히 나아지는 느낌

**4막 전반**: 성교/교감 지속. 죽은 아이를 되찾고 싶은 욕망이 성욕으로 표출되지만 본인은 구체적으로 모름

**4막 중반**: 남편 원혼 방문 → 이름을 불러줌 → 기억 회복 → 남편 성불 → 미련 하나 해소

**4막 후반**: 아이에 대한 갈망 자각 → 명확한 의지를 가진 성교

**4막 끝**: 충족 → 의사 임신 → 성불

### 6-4. 트루 엔딩 구조
- 주인공이 귀신을 붙잡으려 하지만, 함께 있으면 양기 고갈로 주인공이 위험한 상태
- 귀신이 주인공을 위해 스스로 떠나는 선택을 하는 것이 진정한 해방
