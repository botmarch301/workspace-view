## 설계 원칙

1. **수치 비노출** — 모든 내부 수치는 플레이어에게 직접 표시하지 않음 (소지금 제외)
2. **환경 피드백으로 전달** — 캐릭터 외형/배경 변화/귀신 반응으로 상태를 체감
3. **정규화** — 모든 수치는 0~100 스케일. 계산은 내부에서 소수점 처리

## 1. 시간 체계

### 1-1. 하루 구조
| 시간대 | 가능한 행동 |
|--------|-----------|
| **아침** | 세신, 식사, 출근 |
| **낮** | 출근(근무), 핸드폰 |
| **저녁** | 퇴근, 식사, 쇼핑(배달), 청소 |
| **밤** | VR, 핸드폰, 스킨십, 성감 개발 |
| **심야** | 수면 → 가위눌림/젖 물리기 이벤트 발생 가능 |

- **행동 1회 = 1턴**. 시간대별 2~3턴 소화 가능
- 하루 총 **8~12턴** (행동 선택에 따라 유동)
- 수면은 하루의 마지막. 수면 스킵 불가

### 1-2. 보름달 주기
- **15일 = 1주기** (음력 기반 단순화)
- 보름달 당일: 귀신 성욕 최대 → 성교 이벤트 트리거
- 4막 마지막 보름달이 성불/양기고갈 분기점

## 2. 주인공 수치

### 2-1. 체력 (HP)

| 항목 | 값 |
|------|-----|
| **범위** | 0~100 |
| **초기값** | 70 |
| **사망 임계** | 0 → 심장마비 엔딩 |

| 행동 | 변동 |
|------|------|
| 식사 (즉석) | +5 |
| 식사 (배달) | +8 |
| 수면 (정상) | +10 |
| 수면 (가위눌림 실패) | -5~-20 (귀신 접근도에 따라) |
| 출근 | -3 |
| 성교 | -5 |
| 고개 돌리기 실패 | -10 |

**환경 피드백:**
| 범위 | 캐릭터 자세 | 배경 |
|------|-----------|------|
| 80~100 | 등 펴고 서 있음 | 정상 |
| 50~79 | 어깨 처짐 | 채도 약간 저하 |
| 25~49 | 무릎 굽히고 상체 앞으로 | 채도 눈에 띄게 저하 |
| 0~24 | 벽에 기대/앉음 | 바래진 화면 |

### 2-2. 정신력 (SAN)

| 항목 | 값 |
|------|-----|
| **범위** | 0~100 |
| **초기값** | 60 |
| **고갈 임계** | 0 → 체력에 따라 정신병 or 강탈→자살 |

| 행동 | 변동 |
|------|------|
| 수면 (정상) | +5 |
| VR 자위 | +3 (공포 해소) |
| 핸드폰 (시간떼우기) | +2 |
| 가위눌림 (귀신 접근) | -10~-30 (거리에 따라) |
| 고개 돌리기 실패 | -15 |
| 괴현상 목격 | -3~-8 |

**환경 피드백:**
| 범위 | 캐릭터 표정/몸짓 | 배경 |
|------|----------------|------|
| 60~100 | 무표정~여유 | 정상 |
| 30~59 | 두리번거림, 어깨 움츠림 | 간헐적 떨림 |
| 10~29 | 뭔가를 가리는 자세 | 심장박동 떨림 + 노이즈 |
| 0~9 | 떨림, 공황 | 강한 노이즈 + 화면 왜곡 |

### 2-3. 양기 (QI)

| 항목 | 값 |
|------|-----|
| **범위** | 0~100 |
| **초기값** | 100 |
| **사망 임계** | 0 → 양기 고갈 엔딩 |

| 행동 | 변동 |
|------|------|
| 식사 | +2 (회복 보조) |
| 수면 (정상) | +3 (자연 회복) |
| VR 자위 (사정) | -8 |
| 성교 (사정) | -15 |
| 수면 중 귀신이 빨아감 | -10/밤 (4막) |
| 성감 개발 충분 시 | 빨아감 -10 → -3으로 감소 |

**환경 피드백:**
| 범위 | 캐릭터 외형 | 배경 |
|------|-----------|------|
| 70~100 | 건강한 피부톤 | 정상 |
| 40~69 | 창백, 다크서클 시작 | 가장자리 약간 어두움 |
| 15~39 | 눈 움푹, 핏줄 비침 | 비네팅 강함 |
| 0~14 | Phase 3 (미라화) | 상단에서 내려오는 어둠 |

## 3. 귀신 수치

### 3-1. 호감도 (AFFECTION)

| 항목 | 값 |
|------|-----|
| **범위** | 0~100 |
| **초기값** | 0 (1막), 귀신 복원 진행에 따라 상승 |

| 행동 | 변동 |
|------|------|
| 제물 배치 (관심 있는 것) | +3~+8 |
| 제물 배치 (관심 없는 것) | +0 |
| 스킨십 (귀신이 좋아하는 부위) | +2~+5 |
| 스킨십 (귀신이 싫어하는 부위) | -3~-5 |
| 성교 | +5 |
| 라이터 사용 | -10 (트라우마 자극) |

### 3-2. 만족도 (SATISFACTION)

| 항목 | 값 |
|------|-----|
| **범위** | 0~100 |
| **성불 조건** | 70 이상 (+ 지박령 성불 완료 + 보름달) |

- 성감 개발 진행도에 비례하여 상승
- 성교 시 귀신이 절정에 도달하면 +10~+20
- 절정 없이 사정만 하면 +2
- 만족도가 높을수록 양기 빨아가는 양이 줄어듦

### 3-3. 성감 맵 (SENSITIVITY MAP)

각 부위별로 독립적인 성감 수치를 가짐.

| 항목 | 값 |
|------|-----|
| **범위** | 0~100 (부위당) |
| **초기값** | 부위별 상이 (0~20) |

#### 부위 목록 (잠정)
| 부위 | 성감 유형 | 초기값 | 비고 |
|------|----------|--------|------|
| 입술 | A2 성장 | 10 | |
| 귀/귓볼 | B2 점진 | 0 | 처음엔 간지러워함 |
| 목 | A1 미온 | 15 | 약하게 꾸준히 |
| 쇄골 | D2 잠재 | 0 | 특정 조건 후 반응 |
| 유방 (전체) | A2 성장 | 15 | |
| 젖꼭지 | B3 폭발 | 5 | 오래 걸리지만 최강 |
| 배/옆구리 | A3 포화 | 10 | 과하면 둔감 |
| 등/허리 | A1 미온 | 10 | |
| 엉덩이 | B1 전환 | 0 | 임계점 넘으면 확 좋아함 |
| 허벅지 안쪽 | A2 성장 | 10 | |
| 클리토리스 | A4 역전 | 20 | 과하면 통증 |
| 질 내부 | B2 점진 | 5 | 천천히 반응 증가 |
| 항문 주변 | C2 악화 | 0 | 할수록 더 싫어함 |

> **참고**: 부위 배정은 잠정. 플레이테스트 후 조정 필요.

#### 성감 유형별 곡선

**A. 순응형**
```
A1 미온:    ████████░░ (10~30 사이를 느리게 오감)
A2 성장:    ██████████████████ (0→80까지 꾸준히 상승)
A3 포화:    ████████████░░░░ (0→60 상승 후 둔감화)
A4 역전:    ████████████████░░░░ (0→70 상승 후 통증→하락)
```

**B. 개발형**
```
B1 전환:    ░░░░░░████████████ (오래 0 유지 → 임계점에서 급상승)
B2 점진:    ░░░░░████████████ (아주 천천히 상승)
B3 폭발:    ░░░░░░░░░░████████████████ (가장 오래 걸리지만 최고치)
```

**C. 거부형**
```
C1 고정:    ████ (항상 낮은 고정값, 변하지 않음)
C2 악화:    ████░░░ (반복할수록 하락)
C3 트라우마: ████▓▓▓ (특정 조건 시 공포 반응 트리거)
```

**D. 무반응형**
```
D1 둔감:    ░░░░ (항상 0)
D2 잠재:    ░░░░░░░░████████ (조건 충족 전 0, 충족 후 급상승)
```

## 4. 앵커 시스템 수치

### 4-1. 강도 (INTENSITY)
- **범위**: 1~10
- 1 = 가장 느리고 부드럽게
- 10 = 가장 빠르고 세게
- 부위의 성감 유형에 따라 적절한 강도가 다름
  - A4 역전: 강도 7 이상에서 통증 전환
  - B3 폭발: 강도 3~5 구간에서 가장 효율적
  - C2 악화: 강도와 무관하게 부정적

### 4-2. 행동별 기본 자극량
| 행동 | 자극 계수 |
|------|----------|
| 쓰다듬기 | 0.3 |
| 감싸기 | 0.2 |
| 주무르기 | 0.5 |
| 움켜쥐기 | 0.7 |
| 문지르기 | 0.4 |
| 꼬집기 | 0.8 |
| 비틀기 | 0.9 |
| 당기기 | 0.6 |
| 찌르기/넣기 | 1.0 |
| 뽀뽀 | 0.2 |
| 키스 | 0.5 |
| 핥기 | 0.6 |
| 빨기 | 0.8 |
| 성기 문지르기 | 0.7 |
| 삽입 (움직이기) | 1.0 |

**실제 자극량** = 자극 계수 × 강도 × 부위 감도 보정

### 4-3. 절정 시스템
- **흥분도**: 0~100 (세션 내 임시 수치. 성교/성감 개발 시작마다 리셋)
- 흥분도 100 도달 → **절정**
- 절정 시: 만족도 +10~+20, 성감 수치 상승
- 절정 없이 끝나면: 만족도 +2 이하

## 5. 엔딩 분기 조건표

| 엔딩 | 조건 |
|------|------|
| **심장마비** | 체력 ≤ 0 |
| **정신병** | 정신력 ≤ 0 AND 체력 > 30 |
| **강탈 → 자살** | 정신력 ≤ 0 AND 체력 ≤ 30 |
| **악귀 회귀** | 이사 선택 (선택지) |
| **복상사** | 부적 소지 + 보름달 성교 시 양기 급감 |
| **퇴마 (상담)** | 상담 이벤트에서 퇴마 선택 |
| **퇴마 (퇴마사)** | 퇴마사 호출 이벤트에서 선택 |
| **양기 고갈** | 양기 ≤ 0 (4막) |
| **트루 (성불)** | 지박령 성불 완료 + 만족도 ≥ 70 + 보름달 성교 |

## 6. 밸런스 기준점 (잠정)

| 항목 | 목표 |
|------|------|
| **1막 길이** | 5~8일 (공포 누적 기간) |
| **2막 길이** | 10~15일 (복원 과정) |
| **3막 길이** | 10~15일 (접근 + 첫 보름달) |
| **4막 길이** | 15~30일 (성감 개발 + 성불) |
| **총 플레이타임** | 40~70일 (게임 내 시간) |
| **4막 성감 개발 없이 생존** | 약 10일 (양기 100 → 0) |
| **젖꼭지 완전 개발 (B3)** | 약 20~25회 자극 |
| **트루 엔딩 최소 일수** | 약 50일 |
