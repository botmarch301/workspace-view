## 기승전결 4막 구조

| 막 | 이름 | 핵심 | 플레이 느낌 |
|----|------|------|-----------|
| **기** | 공포 | 이사, 악귀의 공포 공격 | 순수 호러 |
| **승** | 정화의 시작 | VR 도피, 정액→복원, 만월 69, 벽 삽입 | 호러→에로 전환 |
| **전** | 접근 | 귀신과의 거리 좁히기, 첫 보름달 성교 | 탐색 + 스킨십 |
| **결** | 성감 개발과 성불 | 양기 관리, 성감 개발, 사연, 성불 | 풀 시뮬 + 서사 |

### 막 경계

| 전환 | 트리거 |
|------|--------|
| **1막 → 2막** | VR 구입 (공포 도피 수단 획득) |
| **2막 → 3막** | 벽에 숨은 귀신에게 삽입 → 귀신 도망 (주인공이 귀신을 인지) |
| **3막 → 4막** | 첫 보름달 성교 (성적 관계 성립) |

---

## 1막: 공포 (기)

**주인공 상태**: 공포에 질린 히키코모리
**악귀 상태**: 검은 그림자, 사람 형태 아님, 성별 불명
**핵심**: 순수 호러. 악귀의 정신 공격에서 살아남기. VR 없음.

### 악귀의 전략
악귀는 사람의 형태가 아닌 **검은 그림자**로 존재한다.
목표는 피해자의 정신을 갉아먹어 **신체 통제력을 강탈**하고 자살로 몰아가는 것.
이 방식으로 이전 여성 입주자들을 죽여왔다.
물리력 행사도 가능하지만 **기운을 크게 소모**하므로 평소에는 정신 공격을 선호한다.

- 생활 중 고개를 돌리면 **뒤에서 검은 그림자가 스쳐지나감**
- 천장에서 내려오는 형체
- 수면 방해 (소리, 온도 변화, 가위눌림)
- 공포와 수면 부족의 누적 → 정신력 약화

### 흐름
- 이사, 정착, 첫 출근
- 밤에 소름끼치는 현상 발생 (검은 그림자)
- 가위눌림 (수면 중 — 몸 풀기 연타. 피해 누적형)
- 고개 돌리기 (일상 중 — 방향키로 저항. 즉시 판정형)
- 공포와 수면 부족이 누적됨

### 공포 이벤트 실패 분기 (가위눌림 · 고개 돌리기 공통)

| 상태 | 결과 |
|------|------|
| 정신력/체력/양기 충분 | 도망 (가위눌림은 알람에 깸). 게임 계속 |
| 체력 높음 + 정신력 고갈 | 정신병 엔딩 |
| 체력 낮음 + 정신력 고갈 | 신체 강탈 → 자살 엔딩 |
| 체력 약함 | 심장마비 엔딩 |

### 엔딩
- **심장마비** — 공포 과다
- **악귀 회귀** — 이사 결심

### 메뉴
- 식사, 세신, 청소, 출근, 핸드폰(쇼핑/배달/시간떼우기), 수면, 인벤토리

## 2막: 정화의 시작 (승)

**주인공 상태**: VR로 공포를 차단. 자위 루틴.
**악귀 상태**: 정신공격 무력화 → 정액에 노출되어 신체 복원 시작
**핵심**: VR 구입 → 자위 → 정액 노출 → 귀신 복원 → 젖 물리기 → 만월 발각 → 삽입

### 흐름
1. 더 이상 못 버티겠다 → **VR 구입** (2막 시작). 인터넷 검색(괴담)도 이 즈음에 연출로 발생
2. VR 자위 중 사정한 **정액이 악귀에 묻음** → 아랫배(자궁) 복원. 괴현상 줄어듦
3. VR 자위 일상 반복 → 귀신 신체 **점진적 복원** (플레이어만 봄)
4. 젖가슴 복원 후, 잠결에 손가락 빠는 주인공에게 **젖을 물림** (주인공 모름)
5. 젖 물리기 매일 반복. 귀신 표정 미세 변화 — 성적 만족인지 모성애 충족인지 애매 (플레이어만 봄)
6. **만월** — 평소와 다르게 69자세 → 귀신 성기를 만지고 빨아줌 → 귀신 놀라서 도망
7. 귀신이 **벽으로 숨지 못한 하체**에 삽입 → 귀신 놀라서 도망. **2막 끝**

### 메뉴
- 1막 + VR 추가

## 3막: 접근 (전)

**주인공 상태**: 귀신을 인지. 이제는 무섭지 않음. 관심.
**귀신 상태**: 수줍고 경계하지만 관심은 있음.
**핵심**: 귀신과의 거리를 좁히는 과정. 스킨십, 제물, 보름달 성교.

### 흐름
- 2막 끝에서 귀신이 도망. **안 나타남**
- **젖 빨기** (자는 척) — 주인공이 알면서 빠는 인터랙션. 혀 좌우 조작. 들키지 않는 범위에서 최대 자극이 목표
  - 성공 시: 젖 분비량 증가 + 작은 신음 (보상)
  - 귀신이 얼굴을 쓰다듬어줌 — 의심스러우면 손 멈춤 (피드백)
- **제물**: 쇼핑으로 구입 → 인벤토리 → 침대 머리맡 단상에 배치. 귀신이 제물에 주의를 뺏긴 틈에 스킨십 가능
- **팜플렛 사건** — 결혼/출산 팜플렛을 귀신이 빤히 쳐다봄
- **불 트라우마 발견** — 우연히 불을 켬 → 귀신의 발작적 거부. 이후 라이터를 방어 수단으로 사용 가능
- **첫 보름달 — 첫 성교** — 발정 → 삽입 → "임신 시켜달라" → 질내사정. **3막 끝**

### 엔딩
- **복상사** — 동료 부적 → 귀신 차단 → 보름달 성욕 폭발 → 사망
- **퇴마 (상담)** — 상담 → 처녀귀신 진단 → 퇴마 부적 → 퇴치
- **퇴마 (퇴마사)** — 퇴마사 호출 → 귀신 퇴치

### 메뉴 추가
- 인벤토리에 아이템 배치(제물) 해금
- 스킨십 (귀신 보일 때만 활성화)

## 4막: 성감 개발과 성불 (결)

**주인공 상태**: 귀신과 성적 관계. 점점 양기 부족.
**귀신 상태**: 밤마다 나타남. 성교 or 수면 중 양기 빨림.
**핵심**: 성감 개발로 양기 관리 + IDC 사건으로 지박령 성불 + 사연 → 성불

### 귀신의 한의 본질
- 남편과의 가정 → **임신/출산에 대한 미완의 욕구**
- "임신 시켜달라"는 남편에 대한 미련이 투영된 것
- 성욕 폭발도 임신 갈망에서 비롯

### 흐름

#### 양기 위기
- 보름달 이후 밤마다 귀신 등장 (라이터를 켰으면 2~3일 후)
- 성교를 안 해도 자는 사이 성기를 빨아서 양기 하락
- 양기 부족은 환경 피드백으로 전달 (캐릭터 자세/피부톤/다크서클 + 화면 비네팅)

#### 성감 개발
- 기존 VR로 **야동 콘텐츠 추가 구매**
  - 귀신과의 성감 개발과 **동일한 시스템** (앵커, 행동, 강도)
  - 게임은 이것이 튜토리얼이라고 알려주지 않음 (암묵적 튜토리얼)
- VR에서 익힌 것을 귀신에게 적용 → **성감 개발**
- 성감 개발 충분 → 귀신 만족도 상승 → 양기 소모 감소 → 생존 안정화

#### IDC 방화 사건과 지박령
- **같은 방화범**이 IDC(인터넷 데이터센터)에 방화를 저지름
- 주인공의 회사 서버가 해당 IDC에 있어 **출장으로 현장에 감**
- 귀가길에 사고 현장 부근에서 아들이 죽은 자리에 꽃을 놓으려던 **노녀(시어머니)**에게 차가 오는 것을 알려줌 → 노녀가 피하고 가볍게 감사 (주인공에겐 사소한 일)
- **지박령(남편)**은 어머니가 죽을 뻔한 것을 목격. 주인공이 구해준 것을 명확히 알고 은혜를 갚으려 집까지 따라옴
- 지박령이 귀신(아내)과 **재회** → 한이 풀리며 **성불** (플레이어만 목격)

#### 성불로의 흐름
1. 남편 귀신 성불을 목격 → **남편의 죽음을 확인**
2. 가족에 대한 미련이 약해짐 → 감정이 열림
3. 처음으로 **사연을 말함** — 과거, 남편, 방화, 한의 본질
4. 이후 양기 위험 해소 (성욕이 과하지 않게 됨)
5. **마지막 보름달** — 발정하지 않음 → 평상시처럼 성교
6. 임신에 대한 한도 풀렸음을 **자각** → **성불**

### 엔딩 분기

| 조건 | 엔딩 |
|------|------|
| 지박령 성불 + 귀신 만족도 충분 + 보름달 성교 | **트루 엔딩 (성불)** |
| 지박령 성불 미완 or 만족도 부족 | **양기 고갈 엔딩** — 양기 소진 사망 |

### 메뉴 추가
- 성감 개발 (앵커 시스템)

## 일상 시뮬레이션 루프

```
[기상] → [세신] → [식사] → [출근]
                                │
                           [회사 이벤트]
                                │
[퇴근] → [핸드폰(배달)?] → [식사] → [청소?]
                                │
                           [밤 자유 시간]
                    ┌───┼───┐───┐───┐
                  [VR] [핸드폰] [귀신] [스킨십]
                                │
                          [수면/이벤트]
```

### 행동과 수치의 관계 (내부, 비노출)

| 행동 | 영향 |
|------|------|
| 식사 | 체력↑, 양기 회복 보조 |
| 세신 | 컨디션↑ |
| 청소 | 집 상태↑ |
| 핸드폰 (시간떼우기) | 시간 소모, 공포↓ |
| 출근 | 수입↑ |
| VR 자위 | 공포↓, 양기↓ |
| 스킨십 | 호감↑, 성감대↑ |
| 성감 개발 | 귀신 만족↑, 양기 소모 감소 |

## 엔딩 요약

| 엔딩 | 타입 | 발생 시점 | 조건 |
|------|------|----------|------|
| 정신병 | 배드 | 1~2막 | 체력 높음 + 정신력 고갈 |
| 자살 | 배드 | 1~2막 | 체력 낮음 + 정신력 고갈 → 신체 강탈 → 자살 |
| 심장마비 | 배드 | 1~2막 | 체력 약함 |
| 악귀 회귀 | 배드 | 1막 | 이사 선택 |
| 복상사 | 배드 | 3막 | 동료 부적 → 귀신 차단 → 보름달 성욕 폭발 |
| 퇴마 (상담) | 노멀 | 3막 | 상담 → 퇴마 부적 |
| 퇴마 (퇴마사) | 노멀 | 3막 | 퇴마사 호출 |
| 양기 고갈 | 배드 | 4막 | 만족도 부족 or 지박령 성불 미완 |
| **성불** | **트루** | **4막** | **지박령 성불 + 만족도 충분 + 보름달 성교** |
